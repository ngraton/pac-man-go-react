## Pac-Man-Go

A running app that uses gamification to turn your neighborhood streets into a pac-man like game world where movement is determined by your geolocation data.

## Technologies Used

Mapbox, Openlayers, Overpass API, React, Django, django-rest-auth, JS

## What I learned

Intigrating thrid party JS libraries into React Apps. Using Geolocation data from mobile browers. Https. Mapping libraries and how to use them together.
